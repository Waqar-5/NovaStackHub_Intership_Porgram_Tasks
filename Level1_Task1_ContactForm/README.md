# Contact Form — Level 1, Task 1

A clean, responsive contact form built with pure **HTML, CSS, and JavaScript** — no frameworks or plugins.

## Features
- Two-panel responsive layout (info panel + form panel)
- Real-time field validation (name, email, subject, message)
- Custom error messages under each field
- Success message animation on valid submission
- Fully responsive — stacks vertically on mobile
- Easy to wire up to a real backend later (EmailJS, Formspree, or your own API) — form data is already collected into a clean object in `script.js`

## Files
```
Level1_Task1_ContactForm/
├── index.html      → page structure
├── style.css       → all styling
├── script.js       → validation + submit logic
└── README.md
```

## How to Run
Just open `index.html` in any browser — no build step, no server needed.
