# Countdown Timer — Level 2, Task 2

A clean, dark-themed countdown timer built with **HTML, CSS, and JavaScript**.

## Features
- Live countdown showing Days, Hours, Minutes, Seconds
- Defaults to counting down to next New Year on load
- Custom event form — set any name + target date/time
- Quick presets: New Year, +1 Hour, +1 Day, +1 Week
- "Countdown ended" message when target time passes
- Fully responsive glassmorphism-style UI

## How to Run
Open `index.html` in your browser. It starts counting down to next New Year automatically. Try the preset buttons or set your own custom date.
