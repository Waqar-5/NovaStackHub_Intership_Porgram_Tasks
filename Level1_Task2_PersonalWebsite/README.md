# 🌐 Personal Portfolio Website

A modern, responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript** as part of the **NovaStack Frontend Development Internship – Level 1 Task 2**.

The website is designed to present my profile, technical skills, featured projects, and contact information in a clean, responsive, and professional way.

---

## 📌 Project Overview

This portfolio serves as my personal website where visitors can learn about me, explore my projects, view my technical skills, and easily connect with me.

The project focuses on responsive design, modern UI, smooth animations, and interactive user experience using only core frontend technologies.

---

## ✨ Features

* 🎨 Modern and responsive user interface
* 📱 Mobile-friendly design
* 🧭 Smooth scrolling navigation
* ⌨️ Animated typing effect
* 🚀 Scroll reveal animations
* 💻 Skills section with animated progress bars
* 📂 Featured projects section
* 📧 Contact section with Email, LinkedIn, and GitHub links
* ✨ Clean layout with attractive color palette
* ⚡ Lightweight and fast loading

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome Icons
* Google Fonts (Poppins)
