# Personal Website — Level 1, Task 2

A responsive personal portfolio website built with pure **HTML, CSS, and JavaScript**.

## Features
- Sticky navbar with mobile hamburger menu
- Animated typing effect in the hero section
- About, Skills (with animated progress bars), Projects, and Contact sections
- Smooth scroll navigation + active link highlighting
- Fully responsive design (mobile, tablet, desktop)
- Pre-filled with sample content based on real project experience — replace placeholders with your own info

## Files
```
Level1_Task2_PersonalWebsite/
├── index.html      → page structure & content
├── style.css       → all styling
├── script.js       → typing effect, menu toggle, scroll effects
└── README.md
```

## How to Run
Open `index.html` directly in your browser — no build tools required.

## Customize Before Submitting
- Replace `hello@example.com` and social links with your real ones.
- Update phone number / location if you want them shown.
- Swap in your real project descriptions and links under the Projects section.
- Replace the "WQ" avatar circle with an actual photo if you'd like (just add an `<img>` tag in place of the div).

## How to Submit for Internship

2. Take screenshots of the hero section, projects section, and mobile view.
